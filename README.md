#  별 헤는밤

#Front-end 소개
- 일상에 지친 사람들이 모여서 얘기하는 커뮤니티 프로젝트입니다
- Vanilla js를 통해서 서버와 rest api를 주고 받습니다.


---

## 개발 기간
- 2025.10.12 ~ 2025.12.11

---

##  기능
- 회원가입 / 로그인 / 로그아웃
- 사용자 정보 조회 및 수정
- 게시글 CRUD
- 댓글 CRUD
- 좋아요 추가 / 취소
- 이미지 업로드

---

##  기술 스택
- **언어:** JavaScript, HTML, CSS (ES6+)
- **프레임워크:** Spring Boot 3.5.x
- **API통신:** Fetch API, RESTful API
- **인증:** 쿠키
